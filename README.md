# Texscript

```html
<script type="text/texscript">
  Page: "Hello World"
</script>
```

![texscript-hello-world-1](https://github.com/user-attachments/assets/d2f2d0e0-5b43-4913-80f2-90fccf7fa790)

With CSS customizations:

```html
<head>
  <style>
    .card {
      width: 105mm;
      height: 74mm;
      padding: 24px;
    }
  </style>
</head>
<script type="text/texscript">
  Page.card: "Hello World"
</script>
```
